## features

- Http usage
- Json placeholder usage
- simple mvvm usage
##
Clone the project:
```bash
  git clone https://github.com/ardaguness/jsonplaceholder-http-service.git
```

Navigate to the project directory:

```bash
  cd app
```

Install the required packages:

```bash
  flutter pub get
```
Run the Proeject:

```bash
  flutter run
```
